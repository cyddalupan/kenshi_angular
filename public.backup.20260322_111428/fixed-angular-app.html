ToyBits PWA

📱 Install as Native App

⚠️ IMPORTANT: Must use Safari browser

  1. Open in Safari (not Chrome)
  2. Tap the Share button ⎋
  3. Scroll down to "Add to Home Screen"
  4. Tap "Add" in top right
  5. Launch from home screen

iOS only allows PWA installation via Safari browser

📱 Install on Android (Use Chrome)

  1. Open in Chrome browser
  2. Tap the three dots ⋮ in top right
  3. Select "Install app" or "Add to Home screen"
  4. Tap "Install" to confirm
  5. Launch from home screen

💻 Install on Desktop

  1. Use Chrome or Edge for best experience
  2. Look for install icon ⎙ in address bar
  3. Click "Install" when prompted
  4. App will open in its own window
