Install as Native App

Get the full native app experience

📱 iOS Installation (Use Safari)

  1. Open in Safari browser
  2. Tap the Share button ⎋
  3. Scroll to "Add to Home Screen"
  4. Tap "Add" to install

🤖 Android Installation (Use Chrome)

  1. Open in Chrome browser
  2. Tap the three dots ⋮ in top right
  3. Select "Install app" or "Add to Home screen"
  4. Tap "Install" to confirm

💻 Desktop Installation

  1. Use Chrome or Edge for best experience
  2. Look for install icon ⬇️ in address bar
  3. Click "Install" button
  4. Launch from Start Menu or Applications
